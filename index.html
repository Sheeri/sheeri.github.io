Build your own O-FISH instance

This site contains instructions on how to build your own instance of the O-FISH app. Click build to get started.

Prerequisites:
  1. MongoDB Atlas Account (free at cloud.mongodb.com)
  2. MongoDB Community Edition for mongorestore to import the sample dataset
  3. npm for building the web application locally
    Optional:
  4. An AWS environment to use for storing photos (S3) and for the cryptographically secure change history (SES). If you do not have those, the app will still run.
  5. access to an Xcode installation and Cocoapods to build the iOS app
  6. AndroidStudio for building the android app

Get started!

If you already have a build, or if you want to just look at the code, the repositories are:
iOS - https://github.com/WildAid/o-fish-ios
Android - https://github.com/WildAid/o-fish-android
Web - https://github.com/WildAid/o-fish-web
Realm code and sample data - https://github.com/WildAid/o-fish-realm
